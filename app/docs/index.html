IPNetDB service

Method Endpoint Description
GET /healthcheck Returns 200/OK and an "ok" response if the service is healthy.
GET /status Returns 200/OK and an JSON response containing details in the currently loaded IPNetDB databases.
GET /ip/[IP address] Returns 200/OK and an JSON response containing details on the specified IP address. Can be IPv4 or IPv6. Returns 404 if the IP is not in the database.
GET /as/[AS number] Returns 200/OK and an JSON response containing details on the specified AS number. Returns 404 if the IP is not in the database.
